
Scripts for generating new "docs.testnet.json" or "docs.mainnet.json" files from the Evrmore node
=================================================================================================

1) To generate "help_cmd_brief_list_cleaned.txt" from the Evrmore node do:

	/home/ubuntu/evrmore_prebuilt_code/evrmore-2.0.0test1-88fc8c74c/bin/evrmore-cli -testnet -rpcuser=xx -rpcpassword=yy help > help_cmd_brief_list_testnet.txt
	
	/home/ubuntu/evrmore_prebuilt_code/evrmore-1.0.5-b0320a173/bin/evrmore-cli -rpcuser=xx -rpcpassword=yy help > help_cmd_brief_list_mainnet.txt

2) To generate "help_cmd_brief_list_cleaned.txt" from "help_cmd_brief_list.txt" do:

	help-cmd-brief-list-to-cleaned.sh help_cmd_brief_list_testnet.txt help_cmd_brief_list_cleaned_testnet.txt

	help-cmd-brief-list-to-cleaned.sh help_cmd_brief_list_mainnet.txt help_cmd_brief_list_cleaned_mainnet.txt

3) To generate "help_cmd_details_list.txt" from "help_cmd_brief_list_cleaned.txt" using the Evrmore node do:

	while IFS="" read -r -u 10 list_item || [ -n "$list_item" ]; do echo "$list_item: " >> help_cmd_details_list_testnet.txt; /home/ubuntu/evrmore_prebuilt_code/evrmore-2.0.0test1-88fc8c74c/bin/evrmore-cli -testnet -rpcuser=xx -rpcpassword=yy help "$list_item" >> help_cmd_details_list_testnet.txt; done 10< "help_cmd_brief_list_cleaned_testnet.txt"

	while IFS="" read -r -u 10 list_item || [ -n "$list_item" ]; do echo "$list_item: " >> help_cmd_details_list_mainnet.txt; /home/ubuntu/evrmore_prebuilt_code/evrmore-1.0.5-b0320a173/bin/evrmore-cli -rpcuser=xx -rpcpassword=yy help "$list_item" >> help_cmd_details_list_mainnet.txt; done 10< "help_cmd_brief_list_cleaned_mainnet.txt"
	
4) To generate "docs.testnet.json"	from "help_cmd_details_list.txt" do: 

	help-cmd-details-to-docs-json.sh help_cmd_details_list_testnet.txt docs.testnet.json
	
	help-cmd-details-to-docs-json.sh help_cmd_details_list_mainnet.txt docs.mainnet.json

5) If this is testnet, then in "docs.testnet.json" replace every "8819" with "18819"
